Projects

Think-Philosophy Group Website

A resource for the philosophy group that I run. Posts are added according to the topic discussed for each session. Admin page is used for CRUD operations, About and FAQ page also included. visit the site here.

Your Personal Recipe Book

Keep track of your personal recipes. Create, update, view and delete recipes and randomly select for when you can’t decide what to cook. The recipe view allows you to strikethrough ingredients and instructions for when you’re shopping or following step-by-step.

Fantasy League Draft Dashboard

A dashboard that presents statistics from your personal league, that takes part in the Premier League's Fantasy Draft. Statistics include, average score, average winning score and hypotheical head to head score. some stats are visualised with graphs.

Text to Speech Converter

Using Python libraries (pyttsx3 and PySimpleGui), the app allows you to enter text to be converted to speech and exported as an mp3 file. File name and location are user specified and the speech rate can be adjusted via given presets.
