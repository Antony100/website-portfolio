Projects

Philosophy Site

A resource for the philosophy group that I run. Posts are added according to the topic discussed for each session. Admin page used for CRUD operations, about and FAQ page also included.

Recipe Book

A recipe book for keeping track of my personal recipes. I can create, update, view and delete recipes and randomly select for when I can’t decide what to cook. The recipe view allows you to strikethrough ingredients and instructions for when you’re shopping or following step-by-step.

Text to Speech Converter

Using Python libraries (pyttsx3 and PySimpleGui), the app allows you to enter text to be converted to speech and exported as an mp3 file. File name and location are user specified and the speech rate can be adjusted via given presets.
